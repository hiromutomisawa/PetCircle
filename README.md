# Pet Circle

## サイト概要
ペットを通じて交流するSNSサイト
フォロー機能、いいね機能、タグ付け機能などを利用し
自分の飼っているペットの写真を共有したり、情報交換をすることができます。

### サイトテーマ
ペットの飼育経験や写真の共有ができるSNSサイト

### テーマを選んだ理由
コロナ禍におけるペット飼育の増加を受け、
ペット飼育の経験を共有できるサイトが欲しいと思ったため

### ターゲットユーザ
ペットの飼育が初めての人
ペットの飼育経験者

### 主な利用シーン
ペットの自慢や、飼育方法などを共有したい時
