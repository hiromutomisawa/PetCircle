# Pet Circle

## サイト概要
ペットを通じたSNSサイト
フォロー機能、いいね機能、検索機能、タグ付け機能を実装予定

### サイトテーマ
ペットの飼育経験の共有できるSNSサイト


### テーマを選んだ理由
コロナ禍におけるペット飼育の増加を受け、
ペット飼育の経験を共有できるサイトが欲しいと思ったため

### ターゲットユーザ
ペットの飼育が初めての人
ペットの飼育経験者

### 主な利用シーン
ペットの自慢や、飼育方法などを共有したい時
